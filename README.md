# Ohce

Implementation inspired by [this kata](https://kata-log.rocks/ohce-kata) used to train outside-in TDD

Eg.:

```sh
$ ohce Pedro
> ¡Buenos días Pedro!
$ hola
> aloh
$ oto
> oto
> ¡Bonita palabra!
$ stop
> pots
$ Stop!
> Adios Pedro
```

## Usage

```sh
pdm run ohce "Your name"
```

## Build

```sh
pdm build
```

## Test

```sh
pdm run test
```
